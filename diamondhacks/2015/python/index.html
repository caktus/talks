Python is open source programming language which is both powerful and expressive while being easy to learn and use. It's syntax emphasizes readability.

Imperative Python


# hello.py
def hello(recipient):
    print('Hello, {}!'.format(recipient))

if __name__ == '__main__':
    hello('World')
                        

$ python hello.py
Hello, World!

Object-Oriented Python


# gretting.py
class Greeting(object):

    def __init__(self, recipient):
        self.recipient = recipient

    def speak(self):
        print('Hello, {}!'.format(self.recipient))


if __name__ == '__main__':
    greeting = Greeting('Diamond Hacks')
    greeting.speak()
                        

$ python gretting.py
Hello, Diamond Hacks!

Django is Python's most popular web framework and takes on much of the same philosophy as Python. 

from django.shortcuts import render


def home(request):
    return render(request, 'home.html', {'title': 'My Homepage'})
At its most fundamental level, Django routes incoming HTTP requests to Python functions/callables which return HTTP response. Those responses might be HTML (like this one) or JSON, XML, etc 

from django.db import models


class Subject(model.Model):
    name =  models.CharField(max_length=200, unique=True)


class Student(model.Model):
    name =  models.CharField(max_length=200)
    major = models.ForeignKey(Subject)


if __name__ == '__main__':
    math, _ = Subject.objects.get_or_create('Mathematics')
    me = Student.objects.create(name='Mark', major=math)
Django also comes with a flexible Object-relational mapper (ORM) for accessing relational databases for the storage of any type of content you which to model. The declarative style helps with readability.
