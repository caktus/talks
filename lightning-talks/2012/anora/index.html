English is confusing

  • One word can mean two completely different things:
    The wind was too strong to wind the sail.
  • When do you use "a" as opposed to "an"?

It's maddening!!!

"A" vs. "An"

"A" goes before words that begin with consonants.

A chicken crossed the road.

"An" goes before words that begin with vowels.

An owl goes hootie hoot.

There are exceptions to these rules.

  • Use "an" before unsounded "h."
  • When "u" makes the same sound as the "y" in "you," or "o" makes the same sound as "w" in "won," then a is used.

Confused yet?

How it Works


{% load an %}

{{ object|an }} {{ object }} says hootie hoot.

// Output

An owl says hootie hoot.

ANORA

How it works


pip install anora

// Template                         
{% load anora %}
{{ object|anora }} laid a large egg.

// Output
An ostrich laid a large egg.
