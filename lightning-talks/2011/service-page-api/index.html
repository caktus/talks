How Does It Solve It?

Service Pages are hidden webpages the browser opens to provide services to other pages the user visits.

Presenter Notes

Source: /home/copelco/projects/talks/lightning-talks/2011/service-page-api/service-page-api.rst

Architecture

  • Website registers the URL of a Service Page, identified by a type
  • Other sites ask the browser to access a service, by type
  • Browser acts as a proxy
  • Requesting page doesn't need to know who it is talking to
